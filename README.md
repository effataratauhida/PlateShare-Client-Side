#  PlateShare – Donate. Connect. Care.

[Live Site Link](https://plateshare-26a35.web.app)

## Description
PlateShare is a web application that allows users to donate and request leftover food efficiently. The platform connects donors with people in need, making food sharing simple and effective.


## Features
- **User Authentication:** Secure login and registration system.
- **Food Management:** Add, update, and manage food donations easily.
- **Food Requests:** Users can request available food and track request status.
- **Interactive UI:** Smooth animations using **AOS**, alerts using **SweetAlert2**, and notifications with **React Hot Toast**.
- **Responsive Design:** Works seamlessly on mobile, tablet, and desktop devices.
- **Live Updates:** Real-time request status and food management.


## Technologies Used
- **React.js** – Frontend library
- **React Router** – For routing
- **AOS** – Animation on scroll
- **SweetAlert2** – Beautiful alerts
- **React Hot Toast** – Notification system
- **CSS / Tailwind CSS** – Styling
- **MongoDB** – Database for storing food and request data
